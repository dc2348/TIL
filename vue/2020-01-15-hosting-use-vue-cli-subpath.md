# 퍼블릭 경로 설정

###### vue.config.js 파일 생성하기
- `vue.config.js` 파일은 추가적인 설정 파일이다.
- `vue.config.js` 파일을 프로젝트의 root 경로에 생성한다.
- `vue.config.js` 파일은 옴션들을 포함한 하나의 object를 export 해야한다.
```
// vue.config.js
module.exports = {
  // options...
}
```

###### publicPath 설정하기
```
module.exports = {
    publicPath: process.env.NODE_ENV === 'production'
    ? '/'
    : '/'
}
```
